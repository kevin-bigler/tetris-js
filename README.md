# tetris-js

It's tetris.

## To Run

```bash
npm install
```

### (A) For live reload

```bash
npm start
```

Click link in the console

### (B) For bundle file generation

```bash
npm run build
```

### (C) For bundle file generation, updating upon saving any files

```bash
npm run watch
```

## Editing

Edit `app/js/app.js`
or other modules loaded by app.js (directly or indirectly)

Edit `app/index.html`

**Enjoy!**
